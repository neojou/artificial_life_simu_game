# Fonts

## Noto Sans TC (Traditional Chinese)

- File: `composeApp/src/commonMain/composeResources/font/NotoSansTC_Regular.ttf`
- Source: [Fontsource Noto Sans TC](https://fontsource.org/fonts/noto-sans-tc) (based on Google Noto)
- License: SIL Open Font License 1.1 (see `NotoSansTC-OFL.txt` if present, or https://scripts.sil.org/OFL)
- Purpose: CJK glyphs for Compose Desktop + Wasm (Skiko default fonts lack Chinese)

Why: Compose for Web/Wasm paints text with embedded fonts and does not use browser CSS fonts.
